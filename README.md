# Arrow Storm

Arrow Storm is a browser-based rhythm game inspired by Dance Dance Revolution. Players use timed keyboard inputs to match falling arrows, earn points, and submit their score to a leaderboard.

## Features

- Browser-based rhythm gameplay
- Keyboard input controls
- Score tracking system
- Increasing score based on player performance
- Leaderboard system for submitted scores
- SQL database connection for storing score data

## Technologies Used

- HTML
- CSS
- JavaScript
- SQL

## Project Purpose

This project was created for an Intro to Cloud Computing course to practice front-end development, database connectivity, and cloud-based application concepts. The project focused on building an interactive web application and connecting it to a SQL-based backend for leaderboard functionality.

## How to Play

1. Start the game in the browser.
2. Watch for arrows appearing on screen.
3. Press the matching arrow keys at the correct time.
4. Continue playing to increase your score.
5. Submit your score to the leaderboard.
